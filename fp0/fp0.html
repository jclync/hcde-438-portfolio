Project Description

This project will be to create a cookie run game where user uses arrow keys to jump to avoid obstacles (like endless runner/ google dino no internet game) with a login feature to track high scores.

Development Process

Step-by-step plan for my development process:

  1. Create new repo for prototype version
  2. Clone to my computer
  3. Start coding prototype versions

  4. Milestone 1 (prototype version)
  5. Make revisions
    a. Use shapes for character and obstacles (infinite runner)
    b. Make the interface look nicer (colors, game character, obstacles, candy, etc.)
  6. Milestone 2 (final version)
